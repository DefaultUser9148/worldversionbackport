A minecraft fabric mod to allow anyone playing older versions of minecraft (pre 26.2 which dont have Vulkan option) to load and play newer worlds which feature the Vulkan renderer in the video options. And also allows to deupgrade worlds, so you can then remove this mod

Optional: ModMenu

## Why?

Minecraft 26.2-snapshot-1 and i think any version after 26.2 (Which introduced the Vulkan renderer) made several changes to any world that was generated in a previous version because the world generation thing is hardcoded or something, but this 'Upgrading' that happens for older worlds almost COMPLETELY making ANY OTHER VERSION unable to load the world correctly or find any of the chunks/player files as it changes a lot of stuff, most notably the regions and playerdata, moving them into subfolders and making any old version regenerate all of the chunks in the root, and unable to find the playerdata.
